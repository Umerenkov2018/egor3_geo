
## Архитектура

### Стек
- Чистый HTML/CSS/JS, без фреймворков
- Chart.js 4.4 (CDN) — для графиков
- LocalStorage — персистентность состояния

### Персистентность
- `saveState()` — debounce 600 мс + события `visibilitychange` / `pagehide`
- `loadState()` — глубокий merge: `Object.assign(def.plzl, saved.plzl)` — новые поля дефолтов не теряются при эволюции модели

### Требования к программе
- сделать современный дизайн в темных тонах
- использовать систему награждения за правильные ответы и за понимание темы
- сделать кнопку для выбора сложности
- сделать кнопку для сброса опыта
